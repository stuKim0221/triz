📚 TRIZ 문제 해결 도구

창의적 문제 해결을 위한 교육용 앱

TRIZ(트리즈)는 러시아의 겐리히 알트슐러가 개발한 체계적이고 창의적인 문제 해결 방법론입니다.

본 앱은 TRIZ 40가지 발명원리와 모순 매트릭스를 쉽게 학습하고 활용할 수 있도록 돕는 교육용 무료 앱입니다.

주요 기능

  • 40가지 발명원리 검색 및 학습
  • 39가지 공학 파라미터 탐색
  • 모순 매트릭스 활용 실습
  • 학습 메모 저장 및 관리
  • 즐겨찾기 기능
  • AI 프롬프트 자동 생성
